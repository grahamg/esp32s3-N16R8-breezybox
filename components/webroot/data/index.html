ESP32-S3 BreezyBox

Welcome to BreezyBox - a Unix-like shell environment running on the ESP32-S3 microcontroller!

Hardware Specifications

  • Chip: ESP32-S3-N16R8 (Lonely Binary Gold Edition)
  • CPU: Dual-core Xtensa LX7 @ 240 MHz
  • Flash: 16MB
  • PSRAM: 8MB
  • Connectivity: WiFi 2.4GHz + Bluetooth 5.0 LE
  • Interface: Dual USB Type-C (power & data)

BreezyBox Features

File System
WiFi Manager
Virtual Terminals
Shell Scripts
ELF Execution
Unix Commands

Available Commands

BreezyBox provides 20+ Unix-like commands including:

  • File Operations: ls, cat, cp, mv, rm, mkdir
  • Networking: wifi, httpd (this server!)
  • System: free, df, date
  • Utilities: head, tail, wc, more
  • Apps: eget user/repo - download and run ELF programs

Installing Apps

Extend functionality by downloading apps from GitHub:

$ eget valdanylchuk/breezyapps
